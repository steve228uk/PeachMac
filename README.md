# PeachMac
Just another peach.cool client for mac

## Download

A recent signed build can be found [here](http://sht.tl/pIceUs)

## Setup

````
git clone https://github.com/steve228uk/PeachMac.git
cd PeachMac
git submodule update --init
pod install
`````
